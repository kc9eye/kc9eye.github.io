\Security

The application security model.

This class defines the security model employed by the applcation. This class should not be used stand alone. Instead it is instantiated by the Instance class. The methods here should rarely if ever be required to be used stand alone. Instead they are called by other methods in the Instance class. Those methods in Instance class should be employed, not the ones here.
