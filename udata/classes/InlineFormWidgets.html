\InlineFormWidgets

Adds inline form elements to the standard FormWidgets

This class contains methods for creating standard framework styled horizontal forms with simple PHP calls. Includes a validation model, utilizing the jquery.validate() plugin.

checkBox()

checkBox(String  $name, Array  $label, String  $value, Mixed  $required = false, String  $helpBlock = null) : Void

Outputs a styled form checkbox to the stream.

If you require a default value for your checkbox, simply call FormWidgets::hiddenInput prior to calling this method. Use the same $name for the hidden input as this checkbox. Unchecked checkboxes will always have the value given in the hidden input $value parameter.

Parameters

String $name

The name given to $_REQUEST
Array $label

Unindexed array of string labels, the first for the label of the section describing the checkbox, the second for the checkbox itself.
String $value

The value of the element if the checkbox is checked
Mixed $required

Optional true or array of jquery.validate() properties
String $helpBlock

Optional helpblock message for the user

Returns

Void
